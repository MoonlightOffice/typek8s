# typek8s

Write Kubernetes manifests in TypeScript.

## Getting Started

Run the following command in your project to set up typek8s environment.

```shell
deno run -A npm:tiged https://github.com/MoonlightOffice/typek8s/template#v0.5.0 <destination-directory>
```

Go to the directory, write manifests in TypeScript, and run `deno task synth`.
